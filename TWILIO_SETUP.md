# מדריך הגדרת Twilio - Phone Trivia Clicker

## סקירה כללית

מדריך זה יעזור לך להגדיר Twilio Trial עם 15$ קרדיט חינמי לבדיקת המערכת.

---

## שלב 1: הרשמה ל-Twilio (5 דקות)

### 1.1 צור חשבון
1. היכנס ל: https://www.twilio.com/try-twilio
2. מלא את הפרטים:
   - שם מלא
   - אימייל (כל אימייל - גם Gmail)
   - סיסמה חזקה
3. לחץ **"Start your free trial"**

### 1.2 אמת את המספר שלך
1. הזן את מספר הטלפון שלך (ישראלי או אחר)
2. בחר **SMS** או **Call** לקבלת קוד אימות
3. הזן את הקוד שקיבלת
4. ✅ **תקבל 15$ קרדיט חינמי!**

---

## שלב 2: קבלת מספר טלפון (2 דקות)

### 2.1 קנה מספר
1. ב-Console Dashboard, לחץ על **"Get a phone number"**
2. Twilio יציע לך מספר אמריקאי - לחץ **"Choose this number"**
3. המספר יופיע בפורמט: `+1 (XXX) XXX-XXXX`

### 2.2 שמור את המספר
**זה המספר שהמשתתפים יתקשרו אליו!**

דוגמה: `+1 (555) 123-4567`

---

## שלב 3: העתקת פרטי חשבון (1 דקה)

### 3.1 מצא את הפרטים
ב-Console Dashboard תראה:

```
Account Info
├── Account SID: ACxxxxxxxxxxxxxxxxxxxxxxxxxxxx
└── Auth Token: [Show] ← לחץ כאן
```

### 3.2 העתק את הערכים
1. **Account SID** - העתק את כל הטקסט (מתחיל ב-AC)
2. **Auth Token** - לחץ על "Show" ואז העתק

**שמור אותם בצד - תצטרך אותם בשלב 5!**

---

## שלב 4: הגדרת Webhook ב-Twilio (3 דקות)

### 4.1 פתח הגדרות מספר
1. ב-Console, לחץ על **"Phone Numbers"** בתפריט השמאלי
2. לחץ על **"Manage"** → **"Active numbers"**
3. לחץ על המספר שקיבלת

### 4.2 הגדר Webhook לשיחות נכנסות
גלול ל-**"Voice Configuration"**:

1. **"A CALL COMES IN":**
   - שנה ל-**"Webhook"**
   - הזן: `https://[השרת-שלך].up.railway.app/webhook/call/incoming`
   - HTTP Method: **POST**

2. **"Call Status Changes":**
   - הזן: `https://[השרת-שלך].up.railway.app/webhook/call/status`
   - HTTP Method: **POST**

3. לחץ **"Save"**

**דוגמה:**
```
https://trivia-server-production.up.railway.app/webhook/call/incoming
```

---

## שלב 5: עדכון משתני סביבה ב-Railway (2 דקות)

### 5.1 פתח Railway
1. היכנס ל-Railway
2. בחר את הפרויקט **server**
3. לחץ על **"Variables"**

### 5.2 עדכן/הוסף משתנים
הוסף או עדכן את המשתנים הבאים:

```bash
TWILIO_ACCOUNT_SID=ACxxxxxxxxxxxxxxxxxxxxxxxxxxxx  # מהשלב 3
TWILIO_AUTH_TOKEN=xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx  # מהשלב 3
TWILIO_PHONE_NUMBER=+15551234567                   # המספר מהשלב 2
WEBHOOK_BASE_URL=https://[השרת-שלך].up.railway.app # כתובת השרת שלך
```

### 5.3 Redeploy
לחץ על **"Deploy"** או חכה ש-Railway יעשה redeploy אוטומטי

---

## שלב 6: אימות מספרי משתתפים (5 דקות)

### 6.1 למה צריך לאמת?
**Twilio Trial מאפשר שיחות רק ממספרים מאומתים** - זה מגבלת Trial.

### 6.2 איך מאמתים מספר?
1. ב-Twilio Console, לחץ על **"Phone Numbers"** → **"Verified Caller IDs"**
2. לחץ על **"Add a new Caller ID"**
3. הזן מספר טלפון של משתתף
4. בחר **SMS** או **Call**
5. המשתתף יקבל קוד - הזן אותו
6. ✅ המספר מאומת!

### 6.3 כמה מספרים לאמת?
- **מינימום:** 2-3 (לבדיקה בסיסית)
- **מומלץ:** 5-10 (לחידון קטן)
- **מקסימום:** תלוי במגבלת Twilio (בדרך כלל עד 10-20)

## שלב 7: בדיקה ראשונה (5 דקות)

### 7.1 צור משתמש מנחה
הרץ בטרמינל:
```bash
node server/create-user.js
```

### 7.2 היכנס למערכת
1. פתח את הקלינט: `https://[הקלינט-שלך].up.railway.app`
2. התחבר עם המשתמש שיצרת

### 7.3 צור חידון
1. צור כמה שאלות עם 2-4 אפשרויות
2. צור חידון חדש והוסף את השאלות
3. הפעל את החידון

### 7.4 התקשר ובדוק!
1. התקשר למספר Twilio (ממספר מאומת)
2. תשמע: "ברוכים הבאים לחידון. לחצו על מספר התשובה שלכם."
3. לחץ על ספרה (1-4)
4. תשמע צליל אישור
5. בדוק שהתוצאות מתעדכנות בזמן אמת במסך!
